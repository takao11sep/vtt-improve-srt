# VTT to SRT Converter - Claude内部完結型

VTTファイルを高精度で校正してSRT形式に変換するツールです。

## 特徴

- ✅ **外部API不要** - Claudeの内部処理のみで完結
- ✅ **自動フォルダ作成** - タイムスタンプで作業フォルダを自動生成
- ✅ **バッチ処理** - 100エントリずつ処理
- ✅ **高精度校正**
  - フィラー除去（えー、あのー等）
  - 自然な日本語への修正
  - 歯科用語の校正
  - タイムコード完全保持
  - 発話者名保持

## 使い方

### 方法1: コマンドライン

```bash
python3 process_vtt.py /path/to/your/file.vtt
```

### 方法2: ドラッグ&ドロップ（Mac）

1. ターミナルで以下を入力（最後にスペース）:
```bash
python3 /Users/takao11sep/claude-project/vtt-improve-srt/process_vtt.py
```

2. VTTファイルをターミナルにドラッグ&ドロップ

3. Enterキーを押す

## 出力

処理が完了すると、`work_YYYYMMDD_HHMMSS` フォルダが作成され、以下のファイルが生成されます：

- `output_batch_1.srt` - バッチ1の結果
- `output_batch_2.srt` - バッチ2の結果
- ...
- `final_output_corrected.srt` - 最終的な統合ファイル ✨

## 校正ルール

### フィラー除去
- えー、あのー、えっと、その、まあ

### 表現の自然化
- なんすね → なんですね
- っすよ → ですよ

### 歯科用語校正
- ほてつ/ホテツ → 補綴
- わいしょうし → 矮小歯
- きょうごうめん → 咬合面

## 動作環境

- Python 3.6以上
- 標準ライブラリのみ使用（追加インストール不要）
